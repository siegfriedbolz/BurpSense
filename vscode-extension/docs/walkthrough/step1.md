# Load BurpSense Bridge Extension

This walkthrough matches **Burp Suite v2026.4.2** (or the same **2026.4** line) and **Cursor 3.3.30** (or Cursor 3 / VS Code on the extension’s `engines.vscode` line). **BurpSense for Cursor** is a fork by **Siegfried-Thor Bolz** of the original **BurpSense** by **Arqsz** ([TheArqsz/BurpSense](https://github.com/TheArqsz/BurpSense)).

1. Open **Burp Suite** (Professional or Community) on **v2026.4.x** (for example **v2026.4.2**)
2. Navigate to the **Extensions** tab
3. Click **Add** button
4. Select the **BurpSense Bridge JAR file** from your filesystem
5. The extension should appear in the extensions list

The bridge extension enables communication between Burp Suite and **VS Code** or **Cursor**.
